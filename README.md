# Project Title: Demand Forecasting and Inventory Optimization.

## 📌 Business Problem

In retail and manufacturing, inaccurate demand forecasts lead to:
- Stockouts (lost sales)
- Overstock (high holding costs)

This project builds an end-to-end demand planning system to:
- Improve forecast accuracy
- Reduce stockouts
- Optimize inventory levels

Dataset: M5 Forecasting (Walmart sales data)
